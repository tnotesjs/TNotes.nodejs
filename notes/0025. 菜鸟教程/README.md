## 1. 🫧 评价

该笔记记录了菜鸟教程上的 nodejs 教程的目录和链接。

可以先不去过分关注教程中的 nodejs 版本问题，其中介绍的大部分内容，以及提供的示例代码都是正确的，还是很适合用作 nodejs 入门的。

该教程中也提供了 nodejs 测试题，可以刷一刷自测一下。

## 2. 🤔 菜鸟教程中的 node 版本是不是太旧了？

教程内容看过一部分，还是写得蛮不错的，版本或许会比较旧。

当前（25.08） nodejs 官方发布的最新版已经是 v24 了，可是文中（第一篇内容 - `Node.js 概念`）打印的 node 版本却是 v4，但是这也只能说明它开始写教程的时间比较早，内容还是在持续更新中的。比如，在介绍 nvm 的时候（`Node.js NVM`），再次打印 node 版本，就是 v18 了。

温馨提示：

- 如果教程中提供的一些程序无法执行，或者执行结果和教程中描述的行为不一致，那么很可能是因版本不同导致的。
- 如果教程中明确说明了示例程序所使用的 nodejs 版本，那么可以按照提示安装对应的版本再试。
- 没有说明，那就自求多福了！
